## О проекте

Проект по разработке модели глубокого обучения для классификации персонажей мультсериала "Симпсоны".

### Проблема
За 30+ лет трансляции сериала визуальный стиль персонажей многократно менялся:
- Разное качество анимации (1989 vs 2020-е)
- Изменения в дизайне персонажей
- Более 40 классов для классификации
- Несбалансированные данные

### Решение
Создание CNN-модели с применением Transfer Learning, способной надежно идентифицировать персонажей независимо от сезона и стиля анимации.


## Технологии
- **Python**
- **TensorFlow** - построение и обучение моделей
- **OpenCV, PIL** - обработка изображений
- **Pandas, NumPy** - работа с данными
- **Matplotlib, Seaborn** - визуализация
- **Google Colab** - обучение на GPU
- **scikit-learn** - метрики и оценка

## Основные этапы

### 1. Анализ и подготовка данных
- Визуализация распределения классов
- Обнаружение дисбаланса данных

### 2. Построение моделей
- **Baseline**: Простая CNN с нуля
- **Transfer Learning**:
  - EfficientNet
  - ResNet18
- **Fine-tuning** верхних слоев

### 3. Оптимизация
- Подбор learning rate (ReduceLROnPlateau)
- Class weights для несбалансированных данных

##  Результаты

| Модель | F1-score (val) | Особенности |
|--------|----------------|-------------|
| Baseline CNN | 0.79 | Базовая архитектура |
| **ResNet18 + Fine-tuning** | **0.96** | **Лучший результат** |
| EfficientNet | 0.90 | + заморозка слоев, дообучение |
